hexo learn

初步学习hexo部署自己的博客！

基本操作

  1. 静态文件生成： hexo g
  2. 内容部署【提交文件到github】： hexo d
  3. 启动本地server： hexo s
  4. 显示hexo版本号： hexo version
  5. 清除本地hexo临时文件： hexo clean
  6. 新建文章： hexo new “page name”

后续学习网站

  1. 官方中文站点：https://hexo.io/zh-cn/docs/
