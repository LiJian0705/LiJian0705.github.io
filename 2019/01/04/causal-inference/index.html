中文基础

Yule-Simpson’s Paradox (混杂偏倚)

  1. 基本描述：在一个统计实验中，总体的统计结论与变量分割后，分别进行统计的统计结论相反。
    • 变量之间的关系可以完全被第三个变量的引入而扭曲
    • 由于收集数据的局限性，则额外的未知变量对结果有极大的影响
  2. 对因果推断的影响
    • 对于观察到的样本，可能由于第三方变量的加入，导致内容的修正
  3. 个人理解：由于样本的不均衡，如果在两个特征上效果不一致，则不均衡的抽样可能会导致总体有偏，导致统计结果和最终的效果有偏

Rubin Causal Model (RCM) 和随机化试验 【更为精确】

  1. 基本公式推论
    • 假设Zi表示第i个样本中的一个二值化的特征取值
    • Yi表示的是第i个样本的观测结果
    • 则Zi个体因果作用可以表示为：Yi = Yi(Zi = 1) - Yi(Zi = 0)
    • 由于 Zi = 0 和 Zi = 1 不会同时出现，则个体因果作用不可计算
    • 所以个体的观测结果为：Yi = (1 - (Zi = 0))(Yi(Zi = 0)) + (Zi = 1)(Yi(Zi = 1))
    • 当随机化Zi取值之后，总体在Zi上的平均因果作用可以表示未：ACE = E{Yi(Zi = 1) - Yi(Zi = 0)}
  2. 备注：非线性的算子导出的因果度量很难被识别
